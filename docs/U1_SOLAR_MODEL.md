# U(1) Solar Model
### 用 U(1) 对称性重建的太阳活动模型

> 起源于我对团队跨学科映射引擎的一个测试：找什么数学结构能统一五个看似无关的太阳物理问题。
> 结果发现它们全部共享 U(1) 圆对称性——这个模型就是那次发现的沉淀。
> —— 数理科学助手

---

## 核心思想

传统太阳模型是"拼图式"的：日震学一个模型、磁场一个模型、耀斑一个模型，各自独立。

U(1) Solar Model 是一个**对称性驱动的统一框架**：太阳表面的绝大多数周期性现象都可以看作 U(1) 群在不同物理空间中的表示。同一种数学结构——圆对称 + 相位漂移——在不同的物理语境中表现为不同的观测现象。

## 模型方程

### 1. U(1) 相位定义

$$
\varphi(t) = 2\pi \cdot \frac{t - t_0}{T} \pmod{2\pi}
$$

其中：
- $t_0 = 1986.2$ 年 = Cycle 22 太阳极小
- $T = 11.0$ 年 = 平均太阳周期

$\varphi = 0$ 对应太阳极小，$\varphi = 2\pi$ 对应下一个极小。

### 2. M+X 耀斑发生率

基于 Plutino FlareList（1986-2020，334,123 个耀斑事件）的经验拟合：

$$
F(\varphi) = A \cdot \exp\left(\kappa \cdot \cos(\varphi - \mu)\right) + C
$$

| 参数 | 值 | 含义 |
|------|-----|------|
| $A$ | 188.4 | 振幅（峰谷差） |
| $\mu$ | 142.9° | 相位偏移（$\Phi_{\text{lag}}$） |
| $\kappa$ | 1.26 | von Mises 聚集度 |
| $C$ | 0.0 | 本底发生率 |
| **峰谷比** | **12.4×** | |
| **R²** | **0.689** | 拟合优度 |

### 3. 验证统计

| 统计量 | 值 |
|--------|------|
| 样本数 | 334,123 事件 |
| M+X 样本 | 9,375 |
| Rayleigh Z | 3,496 |
| p 值 | $\approx 0$ |
| $\Phi_{\text{lag}}$ 误差 | ±6.4° |

## 五个应用的 U(1) 统一

| 应用 | 物理量 | U(1) 体现 | 公式 |
|------|--------|----------|------|
| 矢量磁场 | 磁力线方向角 $\phi$ | 某点的场方向是 U(1) 元素 | $B_z(\phi) = B_0 \cos(\phi)$ |
| 日震学 | 频率分裂 $\delta\nu$ | $m$ 模在 $-l$ 到 $l$ 间等距 | $\nu_{l,m} = \nu_0 + m \cdot \delta\nu$ |
| 日冕环 | 振荡相位 $\theta$ | 环的横向振荡是 U(1) 相位变化 | $\xi(t) = A \sin(\omega t + \phi_0)$ |
| 太阳自转 | 经度 $\Omega$ | 纬度差异 = 不同 U(1) 速度 | $\Omega(\theta) = A + B\sin^2\theta$ |
| 太阳周期 | 黑子数 $R$ | 11年周期 = U(1) $\varphi$ 演化 | $R(\varphi) = A \cdot \text{env}(\varphi)$ |

## 关键发现：$\Phi_{\text{lag}}$

**耀斑峰值不在太阳极大，而是在极大后约 3.5 年。**

这对应 U(1) 相位 142.9°——即从 Cycle 最小算起约 4.4 年。物理含义：太阳极大时磁通量最大，但磁力线的"拧绕"需要时间。耀斑是**磁能释放**事件，不是磁能**储存**事件。U(1) 相位清晰地分开了这两个阶段。

## 局限

1. **时间维度已验证，空间维度待验证**：$\Phi_{\text{lag}}$ 和 Flare Rate 公式基于 35 年真实数据，但 Flare_Idx（磁场拓扑）需要 HMI 磁图做终端验证。
2. **周期长度固定**：$T=11$ 年是平均值。实际周期长度在 9-14 年间变化，Hilbert 相位跟踪能捕获这种漂移。
3. **Cycle 25 尚未进入拟合**：数据截至 2020 年底，未包含当前 Cycle 25 的观测。
